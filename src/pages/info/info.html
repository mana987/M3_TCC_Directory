Comment utiliser l'application?

  • Liste Entreprise (page Accueil)

    • Une liste des entreprises avec les différent compétences informatique
    il est possible de filtrer cette liste avec le bouton filtre

  • Le Filtre (page Accueil)

    • L'application est équipé d'un filtre par compétence pour affiner vos recherches, les filtres peuvent être sélectionner afin d'afficher les résultats que vous souhaiter,
    un tri par compétence d'une liste de professionel informatique

  • Les Favoris (menu Tabs)

    • Une page dédiée à vos professionnels ajoutés à votre liste des favoris
    vous pouvez consulter vos cartes professionnels ou bien les supprimer de votre liste.

  • Google map (menu Tabs)

    • Permet de lancer google map
